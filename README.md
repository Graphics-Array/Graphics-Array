# Graphics-Array
A 2d Graphics library; a wrapper for SDL. It is beginner friendly and easy to use

- write less, do more

## Example 
a program to create an empty window

```c
#include<Graphics[x][y].h>
int main (){
	init("title",640,480);
	// Program logic here
	quit();
	return 0;
}
```
